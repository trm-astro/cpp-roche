equalrv - direct impact computation.

This program plots the Roche lobes and lines of equal radial velocities over it.

q---Mass ratio = m2/m1
vscale---Velocity scale = k1+k2
vstep---Velocity step between equal RV lines
nstep---Number of steps both to negative and positive velocities
r1---Radius of the accretor (units of Rl1)
rdisc---Disc radius (units of Rl1)
phase---Orbital phase
device---Plot device
width---Plot width, inches, 0 for default
x1---Left X limit for plot
x2---Right X limit for plot
y1---Lower Y limit for plot
y2---Upper Y limit for plot
lwidth---Line width, multiple of default
colours---Do you want colours?
